A system built to test the accuracy of an Explainable AI Clinical Decision Support System for the use in the emergency Department of UHWI.
## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
